# Space Spotter (Memory Game)

Space Spotter is a memory game created with React. The goal of the game is to click all 20 images, without clicked the same one twice.

## Ideas Practiced
* React
* Sass
* Functional Components
* useState & useEffect
* CSS & HTML Design
